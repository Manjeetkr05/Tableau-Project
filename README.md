# 📊 Sales Performance Dashboard (Tableau)

## 🎯 Goal
Build an interactive dashboard to help a retail business quickly understand sales, profit, and trends for better decision-making.

---

## 📌 Dataset
- Sample Superstore (built-in Tableau dataset)

---

## 🚀 Dashboard Features

### 🔹 KPI Tiles
- Total Sales  
- Total Profit  
- Profit Margin (%)  

### 🔹 Trend Analysis
- Monthly Sales Trend (Line Chart)

### 🔹 Category Breakdown
- Sales by Category (Bar Chart)

### 🔹 Geographic Analysis
- Sales by State (Filled Map)

### 🔹 Interactive Filters
- Date  
- Region  
- Category  

---

## 🛠 Tools Used
- Tableau Public  
- Data Visualization  
- Business Analysis
